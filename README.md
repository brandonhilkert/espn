ESPN Headline Feed
=========

This is a simple Sinatra/Backbone app that gives you a ESPN news feed. The new feed is nice, but doesn't remember if you've read articles. I want to save this infor so it's easy to recognize when you come back for viewing. This is a work-in-progress.
